## 📌 Descripción del Proyecto

Este sistema web está diseñado para la **gestión integral del Centro de Educación Alternativa**, permitiendo administrar estudiantes, docentes, notas, asistencias y reportes de manera digital, rápida y eficiente.

El objetivo principal es **modernizar el control académico** y facilitar la comunicación entre administradores, docentes y estudiantes.

---

## 🚀 Funcionalidades Principales

✔ Gestión de estudiantes  
✔ Gestión de docentes  
✔ Registro de notas por módulos  
✔ Control de asistencia  
✔ Generación de reportes  
✔ Visualización de ranking académico  
✔ Panel de administración por roles  
✔ Autenticación de usuarios con Firebase  

---

## 🛠️ Tecnologías Utilizadas

- 🌐 HTML5
- 🎨 CSS3 (Responsive Design)
- ⚙️ JavaScript (Vanilla JS)
- 🔥 Firebase (Authentication, Firestore, Hosting)
- 📊 Chart.js (para gráficos estadísticos)
